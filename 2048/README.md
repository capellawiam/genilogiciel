# 2048 Game

## Overview
2048 is a single-player sliding block puzzle game. The objective of the game is to slide numbered tiles on a grid to combine them and create a tile with the number 2048.

## How to Play
1. Use the arrow keys (or swipe on mobile) to move the tiles.
2. When two tiles with the same number touch, they merge into one.
3. Continue merging tiles until you reach the 2048 tile or cannot make any more moves.

## Features
- Simple and intuitive controls
- Engaging gameplay
- Responsive design for mobile and desktop

## Installation
To run the game locally, clone the repository and open the `index.html` file in your web browser.
